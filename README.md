## Overview

The **Bangalore House Price Predictor** is a comprehensive full-stack machine learning application that predicts real estate prices in Bangalore based on various property features. Built with a Flask backend and React frontend, this project demonstrates end-to-end data science and web development skills.

### Key Highlights

- **Machine Learning Model** trained on 13,000+ real property listings
- **Modern UI** with liquid glass glassmorphism design
- **Real-time Predictions** via RESTful API
- **Data Science Pipeline** including data cleaning, feature engineering, and model optimization
- **Responsive Design** works seamlessly on all devices

---

## Features

### Frontend Features
- **Animated Typing Header** - Eye-catching character-by-character animation
- **Liquid Glass Effect** - Premium glassmorphism UI with floating bubbles
- **240+ Locations** - Comprehensive coverage of Bangalore areas
- **Real-time Price Prediction** - Instant results with detailed breakdown
- **Fully Responsive** - Optimized for mobile, tablet, and desktop
- **Error Handling** - User-friendly error messages and validation

### Backend Features
- **Fast API** - Flask RESTful API with CORS support
- **ML Model Integration** - Linear Regression with 84% accuracy
- **Data Validation** - Input sanitization and range checking
- **Model Artifacts** - Saved model and feature columns
- **Error Handling** - Comprehensive exception handling

### Data Science Features
- **Data Cleaning** - Handling missing values and inconsistencies
- **Feature Engineering** - BHK extraction, sqft conversion, price per sqft
- **Outlier Removal** - Statistical methods for data quality
- **Dimensionality Reduction** - Location grouping and one-hot encoding
- **Model Selection** - GridSearchCV with multiple algorithms
- **Cross-Validation** - K-fold validation for robustness
